Spoonflower
================

Ruby on Rails app
-------------

This application requires:

- Ruby 2.1.3
- Rails 4.2.3

Getting Started
---------------

In terminal:

- $ git clone https://github.com/AGoncharenko/spoon.git
- $ cd spoon
- run $ rails s 
- visit localhost:3000 in your favorite browser
- click on any table raw to view details of designs

Tests
-------------------------

To run tests type in terminal:

- run $ rspec
